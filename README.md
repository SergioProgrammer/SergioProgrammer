### 🇬🇧 About me

I come from a marketing background and moved into development because I wanted to build the things I used to brief. Today I run two projects:

- **[SQStudio](https://sqstudio.es)** — web studio focused on landing pages and custom websites.
- **[Procesia](https://procesia.pro)** — we optimize production processes by building custom modules inside our own app.

I'm currently studying **Computer Science** while shipping real products. I work end-to-end: from talking to the client to deploying the thing.

Strongest with **Astro**, **Next.js** and **React + TypeScript** on the frontend, and **Node/Express + Supabase** on the backend. I care about clean UI, fast load times and code that the next person can read without crying.
